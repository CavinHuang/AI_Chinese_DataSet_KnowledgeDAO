# 数据格式
{

'instruction': '用一句话描述地球为什么是独一无二的。\\n\n', 

'input': '', 

'output': '地球上有适宜生命存在的条件和多样化的生命形式。'

}

包含instruction，input和output三部分，注意，input可以为空。
