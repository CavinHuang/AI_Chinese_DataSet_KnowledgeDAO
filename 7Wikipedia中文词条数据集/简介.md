# Wikipedia中文数据集
包含名词、人物、地名等维基百科中文词条介绍（2019年的数据整理），token总数683.6M（汉字数：341,816,586）。

# 数据格式
{

"instruction": "考虑到这个数字序列, 预测模式中的下一个数字.", 

"input": "3,7,11,15", 

"output": "19"

}

包含instruction，input和output三部分，注意，input可以为空。
